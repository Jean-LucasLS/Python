#### <COMMAND> <--> <Description> ####

python -m venv <name> <--> Create a virtual enviroment (usually, <name> is venv)

gcm python -Syntax <--> Visualize the 'python.exe' path

Set-ExecutionPolicy -ExecutionPolicy Bypass <--> Allow scripts being executed at Windows PowerShell

venv\Scripts\activate <--> Activate virtual enviroment
deactivate <--> Deactivates virtual enviroment

pip freeze <--> Shows all frameworks and its versions
pip freeze > requirements.txt <--> Create the 'requirements.txt' file
